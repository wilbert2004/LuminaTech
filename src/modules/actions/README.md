# Módulo de acciones

## Qué hace

Este módulo agrupa las acciones manuales o automatizadas desde la app.

## Estructura

- `hooks/useActions.js`: lógica del módulo.
- `screen/ActionsScreen.jsx`: pantalla de acciones.
- `services/actionsService.js`: creación y consulta de acciones.
- `style/actionsStyle.js`: estilos del módulo.

## Trabajo realizado

- Se reescribió la pantalla con nombres más claros en español.
- Se ordenó el flujo visual con tarjetas y botones de control.
- Se dejaron alias para mantener compatibilidad con imports anteriores.

## Notas

- Revisar imports cuando se agreguen nuevos botones o controles nativos.
