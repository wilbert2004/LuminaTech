# Módulo de dispositivos

## Qué hace

Este módulo gestiona la vista y lógica relacionada con dispositivos.

## Estructura

- `hook/useDevices.js`: lógica del módulo.
- `Screen/DevicesScreen.jsx`: pantalla de dispositivos.
- `Screen/DeviceCard.jsx`: tarjeta individual de dispositivo.
- `service/devicesService.js`: operaciones de dispositivos.
- `style/Devicesstyles.js`: estilos del módulo.

## Trabajo realizado

- Se mantiene el patrón por capas usado en los demás módulos.
- La vista principal y la tarjeta están separadas para facilitar mantenimiento.
- El estilo está aislado en su propio archivo.

## Notas

- Conviene revisar el casing de carpetas y archivos para mantener consistencia.
