# Módulo de sensores

## Qué hace

Este módulo presenta la información de sensores y su estado en la interfaz.

## Estructura

- `hooks/useSensors.js`: lógica para consultar y manejar sensores.
- `screen/SensorsScreen.jsx`: pantalla de sensores.
- `services/sensorsService.js`: operaciones de lectura y consulta.
- `style/sensors.js`: estilos del módulo.

## Trabajo realizado

- Se reorganizó la pantalla para que el contenido no ocupe todo el ancho en desktop.
- Se ajustaron tarjetas y contenedores para una lectura más limpia.
- Se mantuvo la separación entre lógica, vista y estilos.

## Notas

- Verificar que cada pantalla use su propio archivo de estilos.
