# Módulo de perfiles

## Qué hace

Este módulo muestra y permite trabajar con la información del usuario.

## Estructura

- `hook/useProfile.js`: hook del perfil.
- `screen/ProfileScreen.jsx`: pantalla del perfil.
- `services/profileService.js`: consulta de datos del usuario.
- `style/Profilestyle.js`: estilos del perfil.

## Trabajo realizado

- Se rehizo la pantalla con un layout más limpio y centrado.
- Se normalizaron nombres en español para la lógica principal.
- Se conservaron alias para evitar romper imports existentes.

## Notas

- La ruta real del hook es `hook/useProfile.js`.
- El estilo del módulo vive en `style/Profilestyle.js`.
